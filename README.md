# 🧠 Multimodal Sentiment Analyzer

An AI-powered app that analyzes emotions from both **images** and **text** simultaneously!

## 🚀 What it does
- Upload a face image → AI detects emotion
- Type any text → AI analyzes sentiment
- Combines both for multimodal analysis!

## 🛠️ Technologies Used
- Python
- Hugging Face Transformers
- Gradio
- DistilBERT (text sentiment)
- Facial Emotion Detection Model

## 💡 How to Run
pip install transformers gradio torch torchvision Pillow
python app.py
